Pro\+ is a planned tier for labels that want LabelUtils to feel like their own private bot.

<Warning>
  Pro\+ is **not part of the current Pro plan yet**. This page describes what's coming, not what's available today.
</Warning>

## The idea

A Pro\+ server would connect its own Discord bot token, so the bot can use that label's own bot identity instead of the shared LabelUtils identity. This unlocks:

- Custom **bot name**
- Custom **avatar**
- Custom **online status**
- Custom **profile description**
- Custom **invite identity** for that label

## How is this different from Pro branding?

Current Pro branding already customizes:

- Server-specific messages and embeds
- Submit panel captions and colors
- DM templates
- Footers, success messages, and thumbnails
- The bot's **server nickname** (where Discord permissions allow it)

Pro\+ goes further by running a **separate branded bot connection** for the buyer — so the bot's profile is fully theirs across Discord, not just inside their server messages.

## Why isn't it available yet?

Pro\+ requires stronger hosting because every custom bot needs its own:

- Discord gateway connection
- Memory and process overhead

Pro purchases help fund better VPS capacity so Pro\+ can be built and supported properly.

## Security plans

When Pro\+ launches, Discord bot tokens will be treated like passwords:

- Stored securely
- Never shown in public messages or logs
- Removable or replaceable by the server owner at any time
